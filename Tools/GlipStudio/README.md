Description
--------------
GUI tool for image processing on GPU with GLIP-Lib. Requires Qt toolkit, possibly libraw for raw image loading.

Platforms : *NIX, Windows 7+

More on http://glip-lib.net

Compiling
--------------
Using Qt toolchain :

$ qmake && make
